# Concepts

This is the Concepts branch. From here, there are three new branches:
[[Philosophy]], [[Implementation]], and [[Stories]].

If you're new, you should start with the Philosophy branch.
It's the one where you learn about what Trillia really is and why it was made.
This branch is the concepts stripped of grounded processes.
In here, you'll be able to learn the core guiding principles that drive the language forward.
Tutorials and most concepts that are vital to understanding the language live inside this branch.

If you want to learn more about how the language does what it does, then the Implementation branch is where you will go.
This branch is designed to show you implementation details, help you learn how to optimize on most hardware, and give you an explanation of what's happening internally.

The Stories branch is dedicated to telling you stories about Trillia.
The history behind it, the way in which it was developed, and other stories to tell.
