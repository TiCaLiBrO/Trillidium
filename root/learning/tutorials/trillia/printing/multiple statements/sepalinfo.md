### How Does Print Work?

In the previous lesson, you typed `print("Hello World!")`.
The word `print` is actually something called a function.
With it, we can print whatever is inside the brackets.

The `"Hello World!"` text that we put inside the brackets is what gets printed.
If we put some other text in there, `print` would print that instead.

The `"` quotation marks are needed because it's difficult for people and computers sometimes to tell if something is supposed to be text or code.
So we always use quotation marks when something is supposed to be text.
