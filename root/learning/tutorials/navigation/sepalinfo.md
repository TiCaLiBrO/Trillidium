# Course 0: Navigation

In this course, you will learn the basics of efficiently navigating the manual!

To get here, you used "/navigation" to enter this course.
When you start a command with a *forward* slash `/`, it brings you *forward* into the page with the same name that you wrote beside it.
So you already know how to move forward!

## The First Tutorial: Listing Links

This page has a hidden link that you cannot see.

### Task

Type `/` then hit enter.
You will see a link to a hidden page.
Once you see it, type `/name`, replacing `name` with the page's name, and hit enter to enter the page.

> [!NOTE]
> If you're not in SEPAL, you can see this link [/backslash]().
> 
> If you're in SEPAL, you will see the name `backslash` after you type `/`.
