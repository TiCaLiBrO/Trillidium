# Backslash

Now that you're in here, typing `/`, you will notice that there is no way forward.
Don't worry; You're not trapped in here forever.

## The Lesson

If *forward-slash* `/` lets you go *forward*, *back-slash* `\` takes you *back*!

Type `\` and hit enter to go to the previous page.
Once you've done that, do it again to exit the navigation lesson.
You'll have finished the navigation tutorial, and you will know the basics of navigating the manual on your own.
Congratulations!

> [!NOTE]
> The message displayed when attempting to list elements inside a folder that is empty is:
>
> `No path forward. Use \ to go back.`
>
> If you're not in SEPAL, [\\\\](https://github.com/TiCaLiBrO/Trillia/blob/main/root/learning/tutorials/sepalinfo.md) will take you back to tutorials.
